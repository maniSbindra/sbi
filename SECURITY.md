# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please open a
[private security advisory](https://github.com/manisbindra/sbi/security/advisories/new)
on this repository.

You should receive a response within 48 hours. If for some reason you do not,
please follow up to ensure the original report was received.

Please include as much of the following information as possible:

- Type of issue (e.g., command injection, path traversal, etc.)
- Full paths of source file(s) related to the issue
- Step-by-step instructions to reproduce the issue
- Impact of the issue, including how an attacker might exploit it

## Supported Versions

Only the latest release is supported with security updates.
